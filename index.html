Introducción a CSS

¿Que es CSS?

CSS (Cascading Style Sheets) es un lenguaje de diseño web que se utiliza para controlar la presentación visual de las páginas web. Permite definir el estilo de los elementos de una página, como colores, fuentes, tamaño, diseño y posicionamiento. En esencia, CSS se ocupa de cómo se ve el contenido estructurado por HTML.

¿Cual es su propósito?

Separar el contenido del diseño

CSS permite que el contenido de una página web (definido en HTML) se separe del diseño visual. Esto facilita la gestión y actualización del diseño sin afectar el contenido.

Dar estilo a los elementos HTML:

CSS se utiliza para aplicar estilos a elementos HTML, como párrafos, encabezados, imágenes y enlaces.

Controlar la presentación visual:

CSS permite controlar diversos aspectos visuales, como colores, fuentes, tamaño, márgenes, espaciado, fondos, layouts y mucho más.

Optimizar el rendimiento:

Al separar el contenido del diseño, CSS contribuye a optimizar el tiempo de carga de las páginas web.

Hacer que las páginas web sean más atractivas:

Con CSS, los diseñadores pueden crear páginas web con diseños visualmente atractivos.

En resumen, CSS es un lenguaje esencial para el desarrollo web que permite dar estilo y presentación a una página web, creando una experiencia visual atractiva y optimizada.. Si vas a dedicarte a la web en mayor o menor medida tendrás que conocer CSS ya que toda la web está hecha de HTML + CSS.

Características:

  1. Lenguaje de texto plano, se puede desarrollar con cualquier editor
  2. Lenguaje nativo de los navegadores
  3. No necesita ser compilado ni transpilado
  4. No es un lenguaje de programación, es un lenguaje de marcado de la información
  5. Nació con internet

Estructura de una sentencia CSS

La estructura de un documento CSS en realidad no tiene estructura alguna. Es una relación de propiedades una detrás de otra sin orden establecido "a priori". Existen diferentes metodologías a la hora de ordenar las propiedades.

Los documentos de CSS son archivos de texto como HTML pero con la extensión .css.

En el código de un documento .css se escriben una serie de órdenes y el cliente (navegador) las interpreta y aplica a los documentos HTML.

Componentes CSS

Concepto Descripción
Selector Elemento o elementos del documento que vamos a seleccionar para aplicarle un estilo concreto
Propiedad Característica principal que vamos a definir con el valor indicado a continuación
Valor Cada propiedad tiene una serie de valores concretos asignables, con los que tendrá uno u otro comportamiento.
Comentario Fragmento de texto entre /* y */ con anotaciones o aclaraciones para el desarrollador (el navegador los ignorará).
Regla Par de propiedad y su respectivo valor asociado.
